隐私政策

SLE 手账 · v1.0 · 生效日期 2026-05-27

1. 关于我们

SLE 手账 App（以下简称"本 App"）由 Craft with love 二人团队开发与运营，旨在帮助系统性红斑狼疮（SLE）患者整理检验报告、跟踪长期指标趋势。本 App 不是医疗器械，不提供诊断、不预测病情、不推荐用药、不替代医生判断，仅作为患者自我记录与就诊准备的辅助工具。

本隐私政策说明本 App 如何收集、使用、存储与共享你的个人信息。请在使用本 App 前仔细阅读。

2. 我们收集什么数据，以及为什么

2.1 你主动提供的

检验单照片 / 门诊病历照片
用途：识别检验项目、数值、参考区间、医院名、就诊日期。
处理流程：照片上传至本 App 服务器 → 转发至 OCR 服务商进行图像识别 → 识别结果写入你的账号记录。
存储期限：图片会保存在我们的服务器，用于你之后查看原始报告、核对识别结果和删除记录。你删除单份报告或删除账号后，服务器会删除对应图片和相关记录。
替代选择：你可以选择不上传，手动输入数值。
每日打卡数据（症状、用药记录）
用途：本地长期追踪你的病情记录。
处理流程：保存在你的账号记录中，用于跨页面展示、导出和未来数据恢复。
存储期限：直到你在 App 内手动删除或删除账号。
检验数值与观察记录（observations）
用途：本地展示趋势图、指标卡片。
处理流程：OCR 识别完成后，结构化检验项和你确认保存的 observation 会写入服务器数据库，并在 App 中用于展示趋势图、指标卡片和就诊记录。
存储期限：直到你删除对应报告、删除单条 observation 或删除账号。
去标识化识别改进候选
用途：用于人工审核检验项别名库，改进 OCR 后的指标匹配准确率。
包含：检验项目名、单位、标本提示、清洗后的医院名、报告类型、用户手动确认的标准指标、出现次数。
不包含：原图、患者姓名、医生姓名、图片路径、检验数值、参考范围全文。
内测规则：TestFlight 内测用户参与测试即表示同意上述去标识化候选用于人工审核；我们会在内测说明中明确告知。
正式版规则：正式用户可在 App 内开启或关闭“帮助改进识别准确率”，关闭后后续数据不再进入候选导出。
用药清单
用途：提醒服药、记录剂量。
处理流程：保存在你的账号记录中，用于 App 内展示和导出。
存储期限：直到你手动删除或删除账号。

2.2 账号相关数据

邮箱地址（来自 Apple Sign In）
用途：账号识别、跨设备登录。
来源：你授权 Apple Sign In 时，Apple 提供给我们的邮箱（可能是你的真实邮箱，也可能是 Apple 生成的私密转发邮箱）。
处理流程：仅存储在我们的服务器数据库，不分享给任何第三方。
存储期限：账号存续期间。你可以随时在「我的 → 删除账号」彻底删除。
配额计数（OCR 使用次数、PDF 导出次数）
用途：免费版用户每月使用次数限制；订阅用户的订阅有效期。
存储位置：仅存储在我们的服务器。
存储期限：账号存续期间；删除账号后，仅保留不可逆哈希标识与必要交易审计记录，用于防止重复领取免费额度、处理订阅恢复、退款争议和合规审计。
反滥用与支付审计标识
用途：防止删除账号后重复领取新用户赠送额度，避免同一 Apple 订阅被反复重绑以重置当期使用次数，并处理退款 / 订阅争议。
处理流程：我们不会保留原始 Apple 用户标识或原始交易号；服务器只保存经 HMAC-SHA256 处理后的不可逆哈希标识。
存储期限：删除账号后仍会保留一段必要时间，仅用于上述反滥用、支付审计和合规目的，不用于广告、画像或第三方追踪。

2.3 自动收集的数据

本 App 不收集下列数据：

  • 不收集设备位置（GPS、IP 定位）
  • 不收集通讯录、相册以外的相册
  • 不进行跨 App 追踪（No App Tracking Transparency 提示）
  • 不嵌入第三方广告 SDK
  • 不收集任何用于行为画像或广告投放的数据

本 App 会收集下列最小化的技术数据：

  • IP 地址（仅用于 API 请求路由与限流防滥用，不与用户身份长期关联）
  • 设备型号、操作系统版本（用于故障排查，与请求一同短暂出现在服务器日志中，日志保留期 7 天）

  • 在未经你明确同意的情况下，与上述第三方以外的任何机构共享你的数据

4. 数据存储与安全

4.1 存储位置

  • 账号健康记录（检验数值、症状、用药、观察记录）：存储在服务器数据库，用于 App 内展示、导出和账号数据恢复
  • 服务器数据（账号信息、配额）：阿里云北京区 PostgreSQL 数据库
  • OCR 图片：阿里云北京区文件存储，保留至你删除对应报告或删除账号

4.2 安全措施

  • 所有 App 与服务器的通信使用 HTTPS（TLS 1.2+）加密
  • 账号认证使用 JWT，密钥定期轮换
  • 原始图片仅在服务器存放期间，不会通过 API 暴露给客户端
  • OCR 服务调用在服务器侧完成，API Key 不会出现在 App 中
  • 日志不记录检验单原始内容、患者姓名、检验数值等敏感信息
  • 用于识别准确率改进的数据仅限去标识化候选；内测用户默认参与，正式用户可在 App 内自行开启或关闭

4.3 数据泄露

若发生可能影响你权益的数据安全事件，我们将在 72 小时内通过 App 内通知或邮件告知受影响的用户，并向监管机构报告。

5. 你的权利

根据《中华人民共和国个人信息保护法》（PIPL）以及适用法律，你享有以下权利：

  • 查阅与导出：所有本地数据可通过 App 内「导出」功能生成 PDF 或本地 JSON 文件
  • 更正：App 内所有数据均可直接编辑或重新录入
  • 删除：
    • 单条数据：App 内直接删除
    • 所有本地数据：卸载 App 即彻底删除
    • 服务器账号 + 配额：「我的 → 删除账号」，立即生效，不可恢复
  • 撤回授权：可在 iOS「设置 → Apple ID → 密码与安全性 → 使用您 Apple ID 的 App」中撤回 Apple Sign In 授权
  • 撤回质量改进同意：正式用户可在 App 内关闭“帮助改进识别准确率”，关闭后后续数据不再进入别名候选导出；内测用户可停止参与内测或联系邮箱要求停止后续候选导出
  • 限制处理：不使用 OCR 功能，则我们的服务器从不接收你的检验单图片
  • 投诉：如对本 App 数据处理有疑问，可通过下方邮箱联系我们；亦可向所在地网信办投诉

6. 未成年人

本 App 主要面向成年 SLE 患者使用。如未成年人（不满 18 周岁）需使用本 App，应在监护人指导下使用，且监护人同意本隐私政策。我们不会主动收集与未成年人身份相关的特定数据。

7. 隐私政策的更新

我们可能根据法律法规、产品功能变化而更新本政策。更新后的政策将通过下列方式告知你：

  • 本页顶部"生效日期"会更新
  • 重大变更将在 App 内以弹窗形式提示，并需要你重新阅读

请定期查阅本政策。如不同意更新内容，你可以选择停止使用本 App 并删除账号。
